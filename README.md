FinSight Credit Risk Analysis

This project performs credit risk analytics and NPA prediction using the FinSight Bank dataset.

Contents:
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Regression Modelling (OLS, Ridge, Lasso, ElasticNet)
- Business Recommendations

Tools Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Statsmodels
